# Calendar of Calls

A Cal.com-style call booking service. Guests pick a free slot for an event type and
book a 30-minute call; the owner manages event types, availability, and sees all bookings.

**Live app:** https://ai-for-developers-project-386-hrbj.onrender.com

> Hosted on Render's free tier — the first request after a period of inactivity may take
> ~30–60s while the instance wakes up, and the in-memory data resets on each restart.
